# Hapi Build

Hapi Build é um exemplo de API usando algumas funcionalidades do EcmaScript

## Dando inicio

Primeiro você deve clonar o projeto

```sh
  https://github.com/dougtq/hapibuildexp.git
```

### Instalação

```sh
cd hapibuildexp && yarn
```

### Execução

Desenvolvimeto
```sh
yarn debug
```

Produção
```sh
yarn execute
```
